# docker-debomatic
Debomatic with Docker Compose

# How to use:

* run 'docker-compose up -d'
* Put your Debian/Ubuntu source code at incoming directory.
* Wait for the magic
* All packages generated will be stored at incoming/distribution/pool/package-version
